# Kuesioner Posyandu Remaja

Aplikasi kuesioner berbasis HTML/CSS/JavaScript untuk evaluasi efektivitas program Posyandu Remaja dalam meningkatkan kesadaran mental dan reproduksi remaja di Kota Depok.

## Fitur

- ✅ Formulir kuesioner lengkap dengan 7 bagian
- ✅ Validasi form otomatis
- ✅ Logika dinamis berdasarkan jawaban responden
- ✅ Tampilan responsif untuk mobile dan desktop
- ✅ Export data ke format JSON
- ✅ Tidak memerlukan build atau dependencies

## Struktur

```
.
├── index.html          # File utama aplikasi (HTML + CSS + JavaScript)
└── README.md          # Dokumentasi
```

## Cara Menggunakan

1. Buka file `index.html` di browser web modern
2. Isi kuesioner sesuai dengan petunjuk yang diberikan
3. Setelah selesai, klik "Kirim Kuesioner"
4. Data akan otomatis diunduh sebagai file JSON

## Deployment

Aplikasi ini dapat di-deploy langsung ke GitHub Pages tanpa build process:

1. Push file `index.html` ke repository GitHub
2. Aktifkan GitHub Pages di Settings > Pages
3. Pilih source: "Deploy from a branch" > "main" > "/ (root)"
4. Aplikasi akan tersedia di `https://[username].github.io/[repository-name]`
